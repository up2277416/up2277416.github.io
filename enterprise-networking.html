Enterprise Networking

This module focused on the implementation of complex, scalable networks. I was tasked with demonstrating routing and Layer 2/3 networking skills using industry level Cisco hardware and software.

Skills Learned & Applied

Routing and Switching

Configured scalable routing architectures using Multi-Area OSPF. Configured Area Border Routers (ABRs) and 802.1Q trunking and VLAN segmentation. Implemented Router-on-a-Stick and Layer 3 Switch Virtual Interfaces (SVIs) to allow for more efficient communication between isolated network segments. Gained an understanding of Layer 2 and Layer 3 technologies, including Spanning Tree Protocol (STP) to prevent loops and EtherChannel for link redundancy and Border gateway protocol (eBGP).

Practical skills Assesment

Practical Skills Assessment (Achieved a 1st): Tested on my proficiency in network configurations using Cisco Packet Tracer and Cisco networking equipment. I had to manually configure multi-area OSPF, including Area Border Routers (ABRs), Layer 3 switching (SVI), and Router-on-a-stick. I needed to do this across a six router/switch network and ensure that there was network/IP connectivity between the different OSPF areas. In Cisco Packet Tracer, I had to troubleshoot a network with connectivity issues and address the situation within a timed period.

Technical Tools: Cisco IOS, OSPF, BGP, RSTP, Packet Tracer
