Networking Essentials

This module provided a foundational understanding of network architecture and the OSI model. I focused on designing logical topologies, calculating hardware requirements for business divisions, and implementing baseline security on Cisco network infrastructure.

Skills Learned & Applied

LAN Topology & Capacity Planning

Designed a logical VLAN topology for seven different divisions such as a production division using Cisco 2960-24TT switches. Calculated port capacity requirements for 59 devices and implemented Gigabit Switch links to ensure high-speed backbone connectivity.

Cisco IOS Security

Configured core security settings on network switches, including VTY lines for remote management and console access control. Implemented local user authentication to ensure that only authorized administrators can access the CLI.

Inter-vlan routing/IPv4 Addressing

Planned and implemented an IPv4 addressing scheme across the network based on how many devices were required in each division. Managed static IP assignments for end devices. I implemented a Router-on-a-Stick topology by configuring 802.1Q trunking on a single physical Router interface. By making use of virtual sub-interfaces, I created a cost-effective and scalable routing solution that handles traffic for multiple VLANs without the need for additional physical hardware.

Switching Logic & Full-Duplex

Used Ethernet switching to enable full-duplex communication, significantly reducing network congestion and minimizing unnecessary broadcast traffic, improving overall network reliability and performance.

Technical Tools: Cisco IOS, 2960 Switches, Packet Tracer, Straight-through/Crossover Cabling, Cisco 4321 Routers
