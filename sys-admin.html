System Administration and Internetworking

This module focused on the deployment, security, and management of enterprise-level server environments. I gained deep experience in Linux server hardening, automated infrastructure services, and the implementation of secure web architectures.

Skills Learned & Applied

Web Services & Virtual Hosting

Configured Apache (HTTPD) on Rocky Linux 8 to support name-based virtual hosting. Managed document roots, directory permissions, and configuration files to host multiple secure domains on a single server .

SSL/TLS & Certificate Management

Implemented end-to-end encryption using OpenSSL. Generated private keys, created Certificate Signing Requests (CSRs), and deployed self-signed certificates to secure web traffic via HTTPS/Port 443.

DNS Administration (BIND)

Built and managed BIND DNS servers. Configured forward and reverse lookup zones, managed resource records (A, CNAME, MX), and tested resolution using technical tools like dig, nslookup, and host.

Linux Internetworking (FRR)

Transformed a standard Linux server into a functional router using FRRouting (FRR). Configured OSPF dynamic routing via the vtysh shell, enabling the Linux host to exchange routing tables with enterprise network neighbors.

Technical Tools: Rocky Linux 8, FRR, Apache, BIND, OpenSSL, vtysh shell<
